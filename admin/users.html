Ce module gère les données métier des utilisateurs (fiche, organisation, profil, groupes). La création ou la suppression d'un compte de connexion reste gérée exclusivement par Firebase Authentication — voir « + Pré-provisionner » ci-dessous pour préparer la fiche d'une personne avant sa première connexion.

Pré-provisionner une fiche utilisateur

Cette fiche sera automatiquement complétée dès que cette personne se connectera pour la première fois avec la même adresse e-mail via Firebase Authentication.

Sélectionnez un utilisateur dans la liste pour afficher sa fiche complète.
